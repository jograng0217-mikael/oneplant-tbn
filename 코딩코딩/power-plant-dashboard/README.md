# Won Plant Operations Portal

This is a minimalist operations portal for Won Plant field staff.

## Features
- Branding link to Won Plant's web portal.
- Quick link to the KOMIPO Risk Assessment system.
- Modern industrial-grade dark mode UI.

## Deployment
This project is configured for deployment on **GitHub Pages**.

### How to host
1. Create a new repository on GitHub.
2. Upload `index.html`, `style.css`, and `main.js`.
3. Enable GitHub Pages in the repository settings.
